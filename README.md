# Mee Player

A pixel-art desktop music player built with Electron, Vite, and React.

Mee Player is a simple desktop YouTube playlist player. Paste a public YouTube playlist link and enjoy your music with a clean pixel-style interface.

## Features

- Pixel-art music player UI
- Animated record player with spinning vinyl
- Record change animation
- Interactive progress bar
- Marquee scrolling for long song titles
- YouTube playlist support
- Play songs directly from public YouTube playlists
- No YouTube login required
- Theme switching
- Custom frameless desktop window
- Desktop application support

## Getting Started

Clone the repository and run:

```bash
# Clone repository
git clone https://github.com/thetnyantoe/meephonograph.git

# Enter project folder
cd meephonograph

# Install dependencies
npm install

# Run development mode
npm run dev
```
